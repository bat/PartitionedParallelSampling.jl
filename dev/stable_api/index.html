PartitionedParallelSampling.PartitionedSampling — Type
struct PartitionedSampling <: AbstractSamplingAlgorithm

A sampling algorithm that partitions parameter space into multiple subspaces and samples/integrates them independently (Caldwell et al.).

Constructors:

Fields:

    Default: PriorToUniform()

    Default: 10

    Default: MCMCSampling(strict = false)

  • exploration_sampler::BAT.AbstractSamplingAlgorithm

    Default: MCMCSampling(nchains = 30, nsteps = 800, strict = false)

  • partitioner::SpacePartitioningAlgorithm

    Default: 5
